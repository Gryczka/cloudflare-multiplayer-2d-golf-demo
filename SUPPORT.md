# Support

## Questions

For usage and setup questions, start with the [README](README.md). If your question is not answered there, open a GitHub issue.

## Bugs

Use the bug report template and include reproduction steps, browser details, screenshots when relevant, and any console or network errors.

## Feature Requests

Use the feature request template and describe the problem the feature would solve.

## Security

For security issues, follow [SECURITY.md](SECURITY.md) instead of opening a public issue with details.
