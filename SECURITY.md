# Security Policy

## Supported Versions

This reference demo is maintained from the `main` branch.

| Version | Supported |
|---------|-----------|
| `main` | Yes |

## Reporting a Vulnerability

Please do not disclose security vulnerabilities publicly until a fix is available.

Use GitHub private vulnerability reporting if it is enabled for this repository. If it is not enabled, open a minimal GitHub issue requesting maintainer contact and do not include exploit details in the public issue.

## Response Timeline

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

## Disclosure Policy

We prefer coordinated disclosure after a fix is available and users have had reasonable time to update.
